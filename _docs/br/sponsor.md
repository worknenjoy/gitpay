## Para quem é esta página
Este guia é para pessoas ou organizações que querem patrocinar tarefas para que contribuidores possam trabalhar nelas e ser recompensados quando a entrega for aceita.

## O que você ganha ao patrocinar
- Mais velocidade para resolver issues importantes.
- Progresso transparente via workflow do repositório.
- Condições de aceitação e pagamento mais claras.

## Como funciona (visão geral)
1. Uma tarefa é criada e recebe investimento.
2. Um contribuidor se candidata e é atribuído.
3. O trabalho é enviado via Pull Request.
4. Após aceitação, a recompensa é liberada.
