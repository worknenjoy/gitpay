## O que este guia cobre
Para quem entrega trabalho pago (serviços) usando o repositório para validar a entrega.

## Fluxo típico de entrega
1. Combine escopo e critérios de aceitação.
2. Entregue em PRs pequenos quando possível.
3. Valide via review + CI.
4. Marque como aceito quando os critérios forem atendidos.
5. Dispare o pagamento com base na aceitação.
