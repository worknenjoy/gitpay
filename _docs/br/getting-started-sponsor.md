## O que significa patrocinar
Como sponsor, você financia uma tarefa para que alguém tenha incentivo claro para entregá-la.

## Fluxo simples
1. Identifique uma tarefa com valor claro.
2. Garanta que a issue seja bem definida e testável.
3. Financie a tarefa.
4. Acompanhe o progresso via Pull Request.
5. Após aceitação, o pagamento é liberado.
